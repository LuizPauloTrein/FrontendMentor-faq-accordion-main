# ❓ FAQ Accordion

Projeto desenvolvido como parte de um desafio do Frontend Mentor.

## 📌 Sobre o projeto

Este projeto consiste em um componente interativo de FAQ (Frequently Asked Questions), onde o usuário pode expandir e recolher respostas ao clicar nas perguntas.

O foco principal foi praticar manipulação de DOM com JavaScript, controle de estados (abrir/fechar), além de trabalhar responsividade e organização de layout.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Variáveis CSS
- Media Queries (Responsividade)
- JavaScript (DOM e eventos)

---

## 🎯 O que eu pratiquei

- Estruturação semântica com HTML
- Manipulação do DOM com JavaScript
- Controle de eventos de clique
- Lógica para abrir/fechar acordeões
- Organização de código JS
- Estilização de componentes interativos
- Uso de hover e transições
- Responsividade para diferentes telas
